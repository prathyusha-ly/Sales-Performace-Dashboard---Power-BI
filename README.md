# Sales-Performace-Dashboard - Power-BI

# Overview
An interactive Power BI dashboard built to analyze chocolate shipment data, track sales performance, monitor KPIs, and uncover business insights through dynamic visualizations.

## Dashboard
- Executive KPI Cards
- Sales Trend Analysis
- Product Performance
- Regional Analysis
- Salesperson Performance
- Interactive Filters & Drill-down
  
## Key Insights
- Identified top-performing products and regions.
- Tracked sales trends over time.
- Measured salesperson performance.
- Monitored shipment and order performance.
- Supported data-driven business decisions.
  
## Dataset
The dataset includes:
- Sales Data
- Shipment Data
- Product Details
- Salesperson Information
- Regional Data
